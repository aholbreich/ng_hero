## What is ng_hero

Working "Hero" Application from [Angular2.0 tutorial](https://angular.io/docs/ts/latest/tutorial/)

It includes code comments for learning purpose. 
You will find examples of:

* Components 
* Templates
* Services
* (Auto) injection
* Simple and Advanced Routing
